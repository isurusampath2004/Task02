# Laptop Store — Demo Static Site

Open `my.html` in your browser to view the site. This is a simple static demo used for showcasing laptop listings.

Files added:
- `my.html` — main page
- `styles.css` — styling
- `script.js` — product rendering and interactions

No build step required. To run:

1. Double-click `my.html` or open it in your browser.
2. Click "Shop Laptops" to view products.

If you want a server (optional):

```powershell
# from the folder containing my.html
python -m http.server 8000
# then open http://localhost:8000/my.html
```
